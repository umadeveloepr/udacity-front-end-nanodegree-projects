# NEIGHBOURHOOD MAP

### INTRODUCTION
• In this project we have to display locations on a map
• Open index.html in any browser
• Click on the markers to display their infowindow
• My map displays 5 hardcoded locations of Patiala

### FRAMEWORKS USED
• FourSquare API
• Google Maps API
• KnockoutJS
• jQuery
• Materialize

### HOW TO RUN
• Extract the ZIP file.
• Open index.html in any browser
